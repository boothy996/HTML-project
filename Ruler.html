Ruler class

The Ruler is a special servant who is usually set to watch over holy grail wars, with them usually being allowed command seals of their own.

The class abilities of the Ruler servants include Magic Resistance, protecting them from lower level spells, True Name Discernment, which allows them to know the true name, parameters and skills of servants at an encounter, and Gods Resolution, allowing them command seals, usually 2, to use while presiding over a war.
